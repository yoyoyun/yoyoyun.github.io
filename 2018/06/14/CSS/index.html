需要兼容哪些浏览器？

  • 根据用户群体决定
    • 面向普通用户：IE8+、Chrome、Firefox
    • 企业级产品：IE9+、Chrome、Firefox
    • 搜索：IE7+
  • 了解浏览器市场份额
    • 日志分析，自己的产品，如果IE8的用户只占1%，提醒他升级
    • 百度统计、全球NetMarketShare

浏览器不支持时怎么办？

  • 如果低版本浏览器没有这个特性可以接受吗？
    • border-radius不支持时，没有圆角。很多情况下可以接受
    • box-shadow不支持时，没有阴影
  • 可以使用效果稍微差一些的替代方案吗？
    • min-height: 100vh，但IE8以下不支持，这时可以用不完美降级方案min-height: 800px
  • 可以使用一些替代方案吗？
    • opacity: 0.5在IE下用filter: alpha(opacity=50)
  • 可以使用JavaScript让浏览器支持吗？
    • 使用html5shim.js让IE6~8支持新标签
    • 使用DD_belatedPNG.js让IE6支持半透明png图片
  • 更换实现方式

  • 用的时候查一下caniuse等

CSS2属性

  • IE6不支持min/max-width/height
  • IE6不支持position: fixed
  • IE6-7不支持块级元素inline-block
  • IE6-7不支持display:table
  • 兼容IE6-7并不困难

CSS3属性

  • IE6-7不支持
  • IE8部分支持，box-sizing，outline(只在border外面描一圈，不占空间)
  • IE9基本都支持
