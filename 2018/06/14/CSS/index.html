注释

/**

  • 多行注释
    */

// 是js的注释，但是写在属性前面，那么浏览器会找不到这种属性，相当于声明了一个不存在的属性，就不会执行，但仅限于调试页面临时注释用

CSS选择器

  • 选择器用来从页面中选择元素，以给它们定义样式

通配选择器

  • 用的比较少
    • 指所有元素

标签选择器

id选择器

  • #id

类选择器

  • .class
  • 实际项目中用的最多，会把公共样式提出成组件，用class命名

属性选择器

简单属性选择

  • 元素[属性]

根据具体属性值选择

  • 元素[属性=值]或者[属性=值]，完全串匹配

根据部分属性值选择

  • 元素[属性~=值]或者[属性~=值]
  • ~，部分选择，属性有多个值，其中空格分开的值中有一项是指定的值即可
  • ^，开头匹配。选择href属性值以“#”开头的所有元素
  • $，结束匹配。选择href属性值以“.jpg”结尾的所有元素
  • *，包含匹配。选择class属性值中包含子串“icon-”的所有元素

伪类选择器

  • 伪类（pseudo-classes）
  • 基于DOM之外的信息去（比如根据用户和网页的交互状态）选择元素
  • active一定是focus

选择器组合

  • 组合选择器选中的一定是最后一个元素，前面的元素都是设限的

直接组合 EF

  • 中间没有空格，直接组合，既满足E又满足F
  • 组合时，标签选择器放在最前面

后代组合 E F

  • F必须在E的下边，下图会选中article下边所有的p

亲子组合 E > F

  • F必须是E直接的子集

同时为一组选择器定义样式

  • 逗号分隔，构成选择器组
