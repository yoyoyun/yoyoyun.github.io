ubuntu14.04安装apache2+php+mysql

Posted on 2018-04-04

#ubuntu14.04安装apache2+php+mysql
sudo apt-get update

安装Apache2：

sudo apt-get install apache2

安装PHP模块：

sudo apt-get install php

安装Mysql：

sudo apt-get install mysql-server

输入root用户密码

安装phpmyadmin

其他模块安装：

sudo apt-get install libapache2-mod-php

sudo apt-get install libapache2-mod-auth-mysql

上面两个的作用是，apache能够解析php，需要借助这两个模块来找到php engine

sudo apt-get install php-mysql

上面语句是在php操作mysql数据库时用到

sudo apt-get install php-gd

上面语句是GD库

apache2相关配置

apache2基本上所有的配置都在/etc/apache2目录及子目录下完成

apache根目录

/var/www

apache2配置文件

etc/apache2/apache2.conf
这个文件通过包含其他配置文件涵盖了所有的apache2系统配置信息

改变apache2的默认目录

法一：

apache2的默认目录配置在/etc/apache2/sites-enabled/00default文件中

找到DocumentRoot项，将/var/www改为目标目录即可

法二：

不改变默认目录，在默认目录下面建立一个到目标目录的链接
假设目标目录为/home/username/phptest，则建立链接如下
sudo ln -s /home/username/phptest /var/www/phptest
这样就可以通过http://localhost/phptest访问目标目录了

这种方式更好一点，可以多个工作目录并行开发。
